## 🚀 Building

```bash
# Windows
gradlew.bat clean build

# Linux/Mac
./gradlew clean build
```

Output: `build/libs/LoParkour-6.0.0.jar`

## 📋 Requirements

- Java 17+
- Spigot/Paper 1.20.4+
